# Changelog

## 0.1.1

- 侧边栏改用固定分数行，动态数值不再改变显示顺序，并按阶段隐藏无关信息。
- 碎片菜单改为每玩家状态，修复定位点击无响应；新增 10 碎片下界团队传送与失败退款。
- 修复骷髅、凋零骷髅、猪灵和猪灵蛮兵默认武器被召唤 NBT 破坏的问题。
- 每阶段扩展至 8 个事件，总计 32 个，并加入最近三次防重复。
- 四种大灾变改为 90 秒三阶段攻势，完成奖励提高至 5 碎片。
- 精简聊天和侧边栏中的内部状态信息。

## 0.1.0

- 支持 Minecraft Java 1.21.11、数据包格式 94.1。
- 加入四阶段完整通关流程和分档团队任务。
- 加入四种世界命运、灾厄等级与共享命运碎片。
- 加入 16 个阶段事件和 4 个大灾变。
- 加入末影龙三阶段战斗、水晶加载保护与通关战报。
- 迁移随机附魔系统和 7 组世界永久里程碑奖励。
