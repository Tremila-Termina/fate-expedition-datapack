# 命运远征完整玩法

## 四阶段

1. 荒野求生：12/20 分钟内进入下界，奖励 3/1 碎片；超时灾厄 +3。
2. 炼狱夺火：15/25 分钟内取得烈焰棒，奖励 3/1 碎片；超时灾厄 +2。取得烈焰棒后仍保持下界事件池，直到有人返回主世界。
3. 要塞追猎：30/45 分钟内进入末地，奖励 3/1 碎片；超时灾厄 +2。
4. 终末决战：2/3 分钟内清除水晶，奖励 3/1 碎片；超时灾厄 +2。水晶必须先被实际检测到，才允许完成任务。

血月命运令阶段任务额外 +1 碎片，末日命运将阶段碎片翻倍。

## 世界命运

- 丰饶纪元：补给堆叠数量 +50%，灾厄最低 2。
- 血月降临：战斗事件增加精英，阶段奖励额外 +1。
- 失序世界：事件冷却 5–20 秒，25% 概率连锁。
- 末日倒计时：90 分钟通关，阶段奖励翻倍，超时失败。

## 碎片菜单

- 附魔（2）：随机强化购买者的一件合格物品；没有目标则给予经验修补书。
- 定位（5）：仅在下界选择要塞或堡垒遗迹，定位成功才扣费。
- 护命（3）：无事件时灾厄 -1。
- 挑衅（+2）：每阶段一次，灾厄 +1 并立即触发事件。

## 永久里程碑

获得升级、来硬的、钻石、深入下界、可怕的要塞、光辉岁月和隔墙有眼分别弹出全队四选一菜单。多个菜单顺序排队。属性选择会自动应用给后加入玩家；即时物资不补发。

## 停止与卸载

- 停止：结束当前挑战并清理事件生物，永久属性保留。
- 通关：显示战报并停止，永久属性保留。
- 卸载：移除所有永久属性、记分板和触发器。
